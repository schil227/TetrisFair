## Intro
Tetris: Fair Edition (a.k.a. Tetris: 7-Bag) is a patch which I created for the NES game Tetris. 

The patch features one fundemental change: it uses the [7-Bag Random Generator](https://tetris.wiki/Random_Generator) system instead of total randomness. To briefly summarize: a unique set of each of the 7 tetrominos is randomly generated, and pieces are pulled from that set. When all pieces are used, a new set is generated, and so-on. This randomization system is (for the most part) the standard which modern Tetris games use.
